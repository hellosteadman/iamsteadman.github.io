Primus A WordPress theme

A drop-in theme for WordPress, based on Bootstrap 2.3

Fork it on GitHub  

It’s a very simple WordPress theme, based on a flat version of Twitter Bootstrap. The idea of the theme is to get up and running quickly with a serviceable-looking site, then to extend it via child themes.

Features

  • Customise the header image, background colours and navbar
  • Converts WordPress menu classes into Bootstrap classes, rather than fiddle with Bootstrap's CSS
  • Uses Font Awesome icons
